# Voyaging
##About the Project
Voyaging is an application that implements a hotel reservation service. It provides a user scenario as well as an admin scenario.

##Built with
- Core Java
- Spring framework
  - Spring MVC
  - Spring Boot
  - Spring Data JPA
  - Spring Security
  - Spring Email
  - Spring Validator
- Bootstrap, JavaScrip
##Usage
- As a user :
  -  Search for a room vacancy in the hotel between specific date
  -  Book a room.
